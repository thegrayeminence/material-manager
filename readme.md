### Dev Instructions (for running site on localhost):
#### Backend:
Terminal commands for setup:
1. cd server
1. pipenv install && pipenv shell
2. export FLASK_APP=app.py && export FLASK_RUN_PORT=3000 && export FLASK_DEBUG=1 
3. flask run
#### Frontend:
1. cd client
2. npm install
3. npm run dev
